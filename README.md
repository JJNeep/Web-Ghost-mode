# Web Ghost Mode
This is a peice of code, used in the console or as a bookmarklet, that will trick the web page into thinking you always have focus on a webpage preventing please annoying please gain focus popups on you screen.

## Installation
### Console:
1. go to [code](code.js) and copy it
2. open the dev menu and paste the code into the console tab
### Bookmarklet
1. go to [bookmarklet](bookmarklet.js) and copy it
2. create a new book mark and paste the code in in the url page
3. click on the bookmark to activate
